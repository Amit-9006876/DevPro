# DevDetective Pro (Frontend)

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run dev server:
   ```bash
   npm run dev
   ```

3. Open the URL shown by Vite (usually http://localhost:5173)

This frontend calls GitHub's public API directly (https://api.github.com). For higher rate limits, consider using a backend proxy with a GitHub token.
